Réinitialiser le formulaire ?

Tous les champs remplis dans le générateur d'annonces seront définitivement effacés. Cette action est irréversible.

Du texte est déjà présent

Le générateur d'annonces contient déjà du texte — quelqu'un a peut-être oublié de le vider après une précédente fiche. Voulez-vous le réinitialiser avant de créer la fiche de ce bien ? Le type (maison/appartement) et la ville seront renseignés dans tous les cas.
